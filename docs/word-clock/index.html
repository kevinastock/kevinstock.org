Word Clock

February 2025

This clock tells time in words using a 9Ã—9 letter grid, with only three unused lettersâ€”two of which form part of my signature. The entire front is a single 3D-printed piece that acts as the display, LED adapter, light baffle, and diffuser. Achieving that in a single print was tricky, we had to manually clean between every layer to prevent contamination, but it worked. The model was generated with OpenSCAD using my OpenSCAD Python Wrapper.

Internally, a TinyPICO drives a grid of LEDs, fetching time via NTP. Configuration is handled by an accelerometer: tipping the clock adjusts brightness, manually adjusts the timezone, or resets WiFi. The backplate, secured with heat-set inserts, holds the electronics and includes strain relief for the USB cable.

Not everything went smoothly. Printing took many attempts to avoid light leaks, a firmware bug fried two boards, and a couple of finished clocks mysteriously stopped working.
